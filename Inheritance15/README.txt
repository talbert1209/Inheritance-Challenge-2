You have a class called Employee.  You should make it 
an abstract base class, and make its PayEmployee() abstract, too.

You must create two classes that derive from that Employee
base class that calculates the pay due the given employee.

Undoubtedly, you will need to extend the properties of the
derived classes to support the different ways you need to
pay employees.  Both will need to implement the abstract
PayEmployee() method.

Print the name and the amount to pay the employee in a 
Console.WriteLine()
